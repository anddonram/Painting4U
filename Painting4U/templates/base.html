Painting4U

Se desea construir una página web que permita ver las mejores obras de arte de todo el mundo, con parámetros de busca totalmente rediseñados para todo tipo de usuarios. Desde búsquedas por autor y títulos, hasta por palabras claves que definan el cuadro.
Además de todo esto, se tienen a manos las últimas noticias de arte.
